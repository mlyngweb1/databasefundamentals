Objectives

In this lab you will practice using the JOIN clause with the SELECT statement on a new database.

Exercise

  • Download and open the classicmodels script (as an SQL file).
  • The classicmodels database has 5 tables: customer, productlines, product, orders, and orderdetails.
  • Open the SQL script in MySQL Workbench.
  • Open a new tab and save it as Review JOIN Exercise. Enter (and execute) the following commands:
USE classicmodels; /*Select the Movies database*/
SHOW TABLES; /*List all the tables*/

To do:

  1. Describe each table.

  2. Return the customer name, contact name (concatenated) and phone number for all customers. Sort in alphabetical order of customer name.

  3. Return the customer name, contact name (concatenated), addressLine1, addressLine2, city, state, and phone number for all customers from Ireland. Again sort in alphabetical order of customer name.

  4. Return the customer name, order number and order date for all orders. Sort in alphabetical order of customer name and in descending order of order date.

  5. Retrun the customer name and the number of orders made for each customer who has made an order.

  6. Return the number of orders for customer (Baane Mini Imports). Label the output Number of Orders for Baane Mini Imports.

  7. Return the order number for each order made by Baane Mini Imports. Return each order number once only. Sort in ascending order of order number.

  8. Return the product name, description and product line for all products. Sort in alphabetical order of product name.

  9. Return the product name, description, product line and text description for all products that include the term Chevrolet in the product name. Again, sort in alphabetical order of product name.

  10. Return the product name and quantity ordered for all products identified by order number 10124. Again, sort in alphabetical order of product name.

  11. Return the order number, product name and quantity ordered for all products ordered by customer Baane Mini Imports. Sort in ascending order of order number and then product name.

  12. Return the customer name, order number and order date for all customers even if they have placed no orders. Sort in alphabetical order of customer name and in descending order of order date.
